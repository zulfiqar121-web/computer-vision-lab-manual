##  Installing Visual Studio Code (Step by Step)

Visual Studio Code was used as the code editor for this lab. Follow the steps below to install it.

### Step 1: Download VS Code
1. Open your web browser
2. Go to: **https://code.visualstudio.com**
3. Click on the **Download** button
4. Choose the version according to your operating system (Windows, macOS, or Linux)

---

### Step 2: Install VS Code
1. Open the downloaded installer file
2. Click **Next**
3. Accept the terms and conditions
4. Keep the default settings
5. Click **Install**
6. After installation is complete, click **Finish**

---

### Step 3: Open VS Code
1. Open Visual Studio Code from the Start Menu or Desktop
2. VS Code is now ready to use for Python and computer vision programs
